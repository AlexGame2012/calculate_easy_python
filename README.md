# 🧮 Программа-калькулятор

Простой калькулятор для выполнения базовых арифметических операций: сложение, вычитание, умножение и деление.

## 🚀 Запуск программы

Для запуска программы убедитесь, что на вашем компьютере установлен Python версии 3.6 или выше.

1. Скачайте файлы проекта на ваш компьютер.
2. Откройте терминал или командную строку.
3. Перейдите в директорию с файлами программы.
4. Запустите программу командой:

```bash
python calculator_program.py
```

## 💻 Использование программы

После запуска программы вам будет предложено:
- Ввести первое число
- Ввести второе число
- Выбрать операцию (+, -, *, /)

Программа выведет результат на экран.

## 📊 Пример работы программы

```
Введите первое число: 4
Введите второе число: 3
Выберите операцию (+, -, *, /): +
Результат: 7.0
```
